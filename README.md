# 短網址產生器

#### 輸入網址並產生新的連結使用，新連結會導向原始網站。

## Installing

#### 環境

1.  node.js v-10.15.3

2.  專案套件
+ "bcryptjs": "^2.4.3",
+ "body-parser": "^1.19.0",
+ "express": "^4.17.1",
+ "express-handlebars": "^3.1.0",
+ "mongoose": "^5.8.2"
3.  Mongodb


#### 開啟終端機到存放專案本機位置並執行:
```
git clone https://github.com/king27350/AC-URLShortener.git
```
#### 初始化程式

```
npm install
```

#### 執行程式

```
1. 終端機輸入: nodemon run dev
2. 開啟網頁輸入: http://localhost:3000
```
